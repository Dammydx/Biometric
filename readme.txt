│Explanation/brief walk through:
├── index.html              # Welcome Page
├── signup.html             # Sign Up Page
├── login.html              # Sign In Page
├── attendance.html         # Attendance Page
├── admin.html              # Admin Login Page (username: admin, password: admin123)
├── admin-dashboard.html    # Admin Dashboard to view attendance records
├── style.css               # Main CSS file for styling
└── script.js               # Main JavaScript file for interaction
NB:(^-^)
index.html: The welcome page where users can navigate to the Sign In or Sign Up pages.

signup.html: Handles user registration, saving user data to local storage.

login.html: Allows users to log in, with validation against stored credentials.

attendance.html: Allows users to mark their attendance, saving the data to local storage.

admin.html: Admin login page with default credentials (username: admin, password: admin123).

admin-dashboard.html: Displays all attendance records stored in local storage, accessible only after successful admin login.

style.css: Contains all the styles for the pages.

script.js: Contains all the JavaScript logic for handling user sign-up, sign-in, attendance marking, and admin login.

This website is a simple demonstration for a biometric attendance system demo.
